# Big-data-Learning  
计划用一到三年的时间，打造大数据从低阶到高阶的整合资料。其中有开发，有项目文档，有读书笔记的整理等

## 快速部署  
clone 项目到本地,使用idea打开并用maven构建工程



## 使用技术
|技术|说明|官网|  
--|:--:|:--:|

|Hadoop|分布式存储引擎|https://hadoop.apache.org/
